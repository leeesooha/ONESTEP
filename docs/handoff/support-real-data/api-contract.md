# API Contract: GET /api/supports

## Overview

지원사업 목록을 조회하는 엔드포인트입니다. 카테고리 및 지역 필터를 선택적으로 적용할 수 있습니다.

---

## Request

### Endpoint

```
GET /api/supports
```

### Query Parameters

| 파라미터   | 타입     | 필수 여부 | 설명                                                                 |
|-----------|--------|---------|----------------------------------------------------------------------|
| `category` | string | 선택      | 지원 카테고리 필터. 값: `금융`, `주거`, `취업`, `의료`, `교육`, `심리정서`, `법률` |
| `region`   | string | 선택      | 지역 필터. 값: `전국`, `서울`, `경기` 등                                  |

파라미터를 생략하면 전체 목록을 반환합니다.

---

## Response

### HTTP 200 OK

```json
{
  "data": [
    {
      "id": 1,
      "title": "자립수당 지원사업",
      "category": "금융",
      "region": "전국",
      "deadline": "2026-12-31",
      "description": "보호종료 후 5년 이내 자립준비청년에게 월 40만 원 자립수당을 현금으로 지급합니다."
    }
  ],
  "total": 15
}
```

### Response Fields

| 필드          | 타입      | 설명                              |
|--------------|---------|----------------------------------|
| `data`       | array   | 지원사업 객체 배열 (deadline ASC 정렬) |
| `total`      | integer | 반환된 지원사업 건수                  |
| `data[].id`  | number  | 지원사업 고유 ID                    |
| `data[].title` | string | 지원사업 이름                       |
| `data[].category` | string | 카테고리                        |
| `data[].region` | string | 지역                             |
| `data[].deadline` | string | 신청 마감일 (ISO 8601: `YYYY-MM-DD`) |
| `data[].description` | string | 지원사업 설명                  |

---

## 예시 curl 명령어

### 전체 목록 조회

```bash
curl -s http://localhost:8080/api/supports
```

### 카테고리 필터 (주거)

```bash
curl -s "http://localhost:8080/api/supports?category=주거"
```

### 지역 필터 (서울)

```bash
curl -s "http://localhost:8080/api/supports?region=서울"
```

### 카테고리 + 지역 복합 필터

```bash
curl -s "http://localhost:8080/api/supports?category=주거&region=서울"
```
